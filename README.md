## 2. 프로젝트 소개

<p align="center">
    Hodadoc은 환자의 시간 절약, 병원의 원활한 예약 관리를 위해,
</p>

<p align="center">
    누구나 쉽게 병원 예약·접수할 수 있는 웹사이트 기반 병원 예약 시스템입니다.
</p>

<br>

## 6. 역할별 기능

### 6-1. 환자

| 기능 | 설명 |
|------|------|
| **카카오 소셜 로그인** | 카카오 OAuth를 통한 간편 로그인/회원가입 |
| **환자 정보 등록** | 이름, 생년월일, 전화번호 등 진료에 필요한 기본 정보 등록 |
| **병원 검색** | 병원명, 지역, 진료과목, 필터(야간/주말/편의시설 등) 기반 통합 검색 |
| **지도 보기** | 카카오맵 기반 주변 병원 위치 확인 및 거리순 정렬 |
| **병원 상세 조회** | 병원 정보, 진료시간, 의사 목록, 리뷰 확인 |
| **당일 접수** | 원하는 병원에 당일 접수 신청 (운영시간/휴무일/공휴일 자동 체크) |
| **예약** | 원하는 의사/날짜/시간대를 선택하여 진료 예약 |
| **내 접수/예약 관리** | 접수 현황 확인, 접수 취소, 예약 취소 등 |
| **리뷰 작성/수정/삭제** | 진료 완료 후 병원 리뷰 작성, 수정, 삭제 (예약/접수당 1회 작성 가능) |
| **리뷰 신고** | 부적절한 리뷰에 대한 신고 접수 |
| **1:1 채팅** | 병원과의 실시간 1:1 채팅 (WebSocket STOMP) |
| **실시간 알림** | 접수 상태 변경, 예약 확정/취소 시 실시간 알림 (SSE) |
| **마이페이지** | 개인 정보 수정, 회원 탈퇴 |

### 6-2. 병원 관리자

| 기능 | 설명 |
|------|------|
| **회원가입/로그인** | 병원 관리자 계정 가입 및 로그인 (서버 관리자 승인 후 이용 가능) |
| **SMS 인증/비밀번호 재설정** | SMS 인증번호를 통한 본인 확인 및 비밀번호 재설정 |
| **병원 정보 관리** | 병원명, 주소, 전화번호, 진료과목, 소개글, 이미지, 해시태그 등록/수정 |
| **운영시간 관리** | 요일별 진료시간, 점심시간, 휴무일 설정 |
| **접수 마감 토글** | 당일 접수 마감/재개 설정 |
| **예약 승인 모드 변경** | 예약 자동 승인/수동 승인 모드 전환 |
| **의사 관리** | 소속 의사 정보 등록/수정/삭제 |
| **의사 스케줄 관리** | 의사별 근무 요일/시간, 예약 간격 설정 |
| **의사 휴무 관리** | 의사별 휴무일 조회, 단건/일괄 등록, 일괄 삭제, 월별 초기화 |
| **공휴일 관리** | 병원에 공휴일 적용/제거 |
| **접수 관리** | 당일 접수 목록 확인, 접수 상태 변경 (대기 → 진료중 → 완료) |
| **예약 관리** | 주간 캘린더 조회, 예약 승인/거절/취소, 진료 완료 처리, 슬롯 블록/해제 |
| **통계** | 대시보드 요약, 의사별 랭킹, 일별 트렌드 조회 |
| **리뷰 관리** | 병원 리뷰 확인, 리뷰 통계 조회 |
| **1:1 채팅** | 환자/관리자와의 실시간 1:1 채팅 (WebSocket STOMP) |
| **실시간 알림** | 새 접수/예약 발생 시 실시간 알림 (SSE) |

### 6-3. 서버 관리자

| 기능 | 설명 |
|------|------|
| **로그인** | 서버 관리자 계정 로그인 |
| **대시보드** | 승인 대기 병원 수, 신고 리뷰 수 등 전체 현황 요약 |
| **병원 가입 승인** | 신규 병원 가입 요청 검토 및 승인/거절 |
| **신고 리뷰 관리** | 신고된 리뷰 확인 및 삭제 승인/신고 반려 처리 |
| **기초 데이터 관리** | 진료과목, 필터(야간/주말/편의시설 등) 카테고리 생성/삭제 |
| **1:1 채팅** | 병원 관리자와의 실시간 1:1 채팅 (WebSocket STOMP) |
